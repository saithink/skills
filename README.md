# SaiAdmin Skills

这是 SaiAdmin 高性能 PHP 框架的开发技能。

## 安装

### 方式一 通过 npx skills 进行安装

```bash
npx skills add https://github.com/saithink/skills --skill saiadmin6.x
```

### 方式二 直接在 `github` 上拷贝到本地，放入对应项目的 `skills`
